Live App

The backend (Flask + TensorFlow) requires a proper Python host and cannot run on GitHub Pages.

To try the live app:

  1. Deploy on Railway (free): push repo → railway.app → New Project
  2. Or run locally: pip install -r requirements.txt && python app.py --no-ssl

Opens at http://localhost:5000
